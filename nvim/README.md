Dotfiles about Neovim configuration
---
> This folder include own setting files about Neovim to make PDE(Personalized Development Environment).
 
## Nvim File Structure
Structure is as follows.
```
../.config/nvim
├── init.lua
└── lua/
   └── USER_PROFILE/
      ├── lazy.lua
      ├── mappings.lua
      └── plugins/
         ├── lsp/
         |  ├── lspconfig.lua
         |  └── mason.lua
         └── themes/
```
